# Binary Calculator

A simple **Binary Calculator Web Application** built using **HTML, CSS, and JavaScript**.
This calculator allows users to perform arithmetic operations on binary numbers easily through a clean web interface.

---

## 📌 Overview

Binary Calculator is a lightweight web-based tool designed to perform operations on **binary numbers (0 and 1)**. It helps students and developers understand binary arithmetic used in computer systems.

Binary numbers are the foundation of digital electronics and computing, and this project provides an interactive way to experiment with them.

---

## 🚀 Feature

* Perform **Binary Addition**
* Perform **Binary Subtraction**
* Perform **Binary Multiplication**
* Perform **Binary Division**
* User-friendly calculator interface
* Real-time results
* Responsive design for desktop and mobile

Binary calculators allow users to compute arithmetic operations directly on binary numbers instead of converting them to decimal first. ([GitHub][1])

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/srikesav09/BinaryCalculator.git
```

2. Navigate to the project folder

```
cd BinaryCalculator
```

3. Open **index.html** in your browser.

The calculator will start working immediately.

---

## 💻 How to Use

1. Enter **binary numbers (0 or 1)** using the calculator keypad or keyboard.
2. Select an operation:

   * `+` Addition
   * `-` Subtraction
   * `*` Multiplication
   * `/` Division
3. The calculator will display the result in binary format.
