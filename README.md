# 🔢 Binary Calculator

A simple and interactive **Binary Calculator** built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Perform binary addition, subtraction, multiplication, and division
* Clean and simple UI
* Real-time result display
* Beginner-friendly project

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
